淺漬梅酒

比例

  • 酒:梅子:冰糖=4:2:1 (體積)

以釀23L空罐為例

12L酒+6L青梅+3L冰糖=21L

建議保留一些空間不要裝滿

青梅密度約
6000ml x 0.9g/m = 5400g;5400g / 600g = 9斤，約需9斤青梅

冰糖密度約
3000ml x 2.5g/ml = 7500g 約需7.5kg冰糖

步驟

  1. 容器清洗乾淨，用酒精消毒
  2. 青梅用牙籤去掉蒂頭，有黃色軟爛的部分要切掉
  3. 清洗青梅然後風乾

    乾燥時表面絨毛較白

  4. 將青梅、酒倒入空罐
